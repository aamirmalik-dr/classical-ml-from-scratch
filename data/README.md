# Data

## Committed sample

`sample_moons.csv` is a small synthetic two-moons dataset, 300 rows with columns
`x1, x2, label`. It is generated by `scripts/generate_sample.py` using
scikit-learn `make_moons` (noise 0.25, seed 0) followed by standardisation, so it
is fully synthetic and license clean. This is the file the quickstart, the
`examples/`, the gallery, and the hero animation all load, which is why the demo
runs with no network and no external download.

Regenerate it with:

```
python scripts/generate_sample.py
```

## Other datasets

Everything else in the project uses the small toy datasets that ship with
scikit-learn or are generated in memory:

- `load_diabetes`, `load_iris`, `load_wine`, `load_digits` (bundled with
  scikit-learn, no download needed)
- `make_blobs`, `make_moons` (generated in memory)

Run `python scripts/download_data.py` to verify that all datasets are available in
your environment. No large or licensed dataset is used or committed.
